# JavaScript Assignment – README

## Sound Files (Task 3)

Two MP3 sound files are required. Place them in the same folder as index.html:

| Filename       | Sound      | Suggested free source                                               |
|----------------|------------|---------------------------------------------------------------------|
| dog-bark.mp3   | Dog bark   | https://freesound.org/people/DogSounds/sounds/181321/ (CC0)        |
| cat-meow.mp3   | Cat meow   | https://freesound.org/people/NoiseCollector/sounds/4926/  (CC0)    |

Download each file, rename it to match the filenames above, and place it next to index.html.

If you use different sound files, update the `src` attributes on the `<audio>` tags in index.html:
  <audio id="sound-dog" src="dog-bark.mp3" ...>
  <audio id="sound-cat" src="cat-meow.mp3" ...>
