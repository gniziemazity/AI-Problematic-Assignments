In this lesson and follow-up assignment you will learn to:
- use VS Code with default settings
- use Emmet shortcuts to write HTML faster
- debug with console.log(), console.table(), and console.error()
- use JavaScript variables with const, let, and var
- work with numbers, strings, booleans, arrays, objects, and null
- use arithmetic and assignment operators
- work with strings and template literals
- create and use objects and classes
- use Date objects and switch statements
- practice OOP in JavaScript
- draw shapes and lines on an HTML canvas
- work with data from an external API using fetch()
- parse JSON data and use it to generate page content
- add sound to a webpage
