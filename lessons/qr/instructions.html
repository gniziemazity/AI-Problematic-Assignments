In this lesson and follow-up assignment you will learn to:
- define and use CSS variables with :root and var()
- position elements using CSS
- center positioned elements with transform: translate()
- use viewport units and border-radius
- toggle visibility and button text with JavaScript
- load a third-party JavaScript library from a CDN
- start, stop, and handle callbacks from a QR code scanner library

- create a GitHub account and repository
- enable GitHub Pages to publish a static website
- add a :hover pseudo-class to button elements
