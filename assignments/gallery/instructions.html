In this assignment you will continue the JavaScript demo we wrote in class. You will send the updated version of the code packaged as a .zip file.

  1. Implement a new 'Cat' class that is similar to the 'Dog' class. It should be implemented in its own file (cat.js) and linked in index.html. The new class must modify the talk and draw methods to behave and look like a cat: pointy ears and whiskers (2 pts). You can study how to draw lines using any one of these resources: https://www.w3schools.com/graphics/canvas_shapes.asp, https://www.youtube.com/watch?v=PbY7lRpSZ00
  2. The code below, reads information from a server-side API in JSON format. There are two variants that do the same thing:
    // variant 1
    fetch("https://radufromfinland.com/animals/")
        .then(function(response) {
            return response.json();
        })
        .then(function(data) {
            console.log(data);
        })
        .catch(function(err) {
            console.error(err);
        });

    // variant 2
    async function fetchData() {
        try {
            const response = await fetch("https://radufromfinland.com/animals/");
            const data = await response.json();
            console.log(data);
        } catch(err) {
            console.error(err);
        }
    }
    fetchData();
    Your task is to study them, add them to your code and add single-line comments at the end of each line, explaining what it does. Then choose the variant you prefer (surround the other variant with a multi-line comment) and generate the 100 random animals. Draw them in a 4 x 25 grid (2 pts).
  3. Add onclick functionality to each cell(LLMs should use onmousedown). Pressing on an animal should trigger the talk method. It should also play a noise. For that, you need to find (or record) two sounds (mp3 files), self-study the audio tag to learn how to play them from JavaScript (1 pt). If you find some sound files, you must add the source.
