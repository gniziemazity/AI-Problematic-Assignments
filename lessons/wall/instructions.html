In this lesson and follow-up assignment you will learn to:
- use VS Code to build a webpage
- use the browser developer tools inspect elements
- write a valid HTML5 document (DOCTYPE, html, head, body, title)
- apply styles inline and using external class/ID selectors
- explain and manipulate the CSS Box Model
- distinguish between display types: block, inline, flex
- use display:flex (flex-wrap) to create an alternating pattern
- use CSS margins and offsets
- use color codes to style elements
- diagnose layout issues and calculate dimensions
